Early Web Communication

The first version of HTTP was created by Tim Berners-Lee in 1989. It was extremely simple and allowed a browser to request a single document from a server.

HTTP 0.9

  • Released in 1991
  • Only supported one method: GET
  • Returned only plain HTML
  • No headers, metadata, status codes, or content types

The Growth of the Web

As websites grew in complexity, the protocol had to evolve. Browsers needed images, styling, different content formats, and persistent connections.

HTTP 1.0 Era

  • Introduced in 1996
  • Supported request headers and status codes
  • Allowed different file types such as images and CSS
  • Still opened a new connection for every request

HTTP 1.1 Update

  • Released in 1997
  • Persistent connections became the default
  • Introduced pipelining
  • Became the most widely used version for 15+ years
